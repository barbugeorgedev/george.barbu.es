# Web App (Next.js)

This is the web application for the george.barbu.es monorepo, built with Next.js and maintained by George Barbu.

## Getting Started

**Requirements:**
- Node.js
- Yarn v1.22.19

Install dependencies from the monorepo root:
```sh
yarn
```

## Development

To start the web app in development mode from the root:
```sh
yarn web:dev
```

## Build

To build the web app from the root:
```sh
yarn web:build
```

## Structure

- Source code: `src/`
- Pages: `src/app/`
- Components: `src/components/`
- Styles: `src/styles/`
- Templates: `src/templates/`
